随着数字化的兴起，在与物流行业紧密相关的零售，电商等行业造成了巨大的影响，物流业也正在经历前所未有的变局。近年来，全球供应链问题打乱了物流业前进的步伐。经济形态呈现的新态势要求物流业的复苏，不能再是简单粗暴，而应该更具有成本效益。

由于新冠造成的地区封锁以及其他的地缘政治因素，全球商品的供应和流通受到了非常大的影响。在供应链行业，人们开始关注物流管理软件能带来的效率提升。通过技术的方式提高企业效率，减少对利润率的影响同时还能节省大量成本。企业可以非常容易地管理和跟踪货物运输情况，并改善客户体验。

企业已经在使用仓库管理系统来自动化库存管理，优化存储空间并加快订单的交付。然而，在不确定性很高的时代，弹性供应链并不仅仅是指一系列自动化的流程，企业需要供应链端到端的透明度管理，这样能快速发现和纠正运营效率低下的问题。

## 何为物流自动化

![何为物流自动化](https://cdn.abmcode.com/zh-cn/industry/_media/logistics/logistics_2.svg) {.center .size-3 .float-l}

我们首先回答是什么的问题，什么是物流自动化？

物流自动化能避免手动执行重复性高的任务，并支持对流程的高效管理，从而降低人为错误的风险。

从运营的角度来看，物流自动化主要是简化流程，提高透明度、准确性和效率，以标准化质量管理并提高服务水平。可以涉及许多技术领域，例如无人驾驶运输和智能运输中心（例如港口和仓库）。

具有物流自动化战略思考的企业，可以成功地将物流管理软件集成到企业的整个数字生态系统中。而正是这种数字优先的业务，才能真正实现业务流程转型和自动化，提高竞争门槛。


## 为何物流自动化

![为何物流自动化](https://cdn.abmcode.com/zh-cn/industry/_media/logistics/logistics_3.svg) {.center .size-4 .float-r}

上面我们回答了是什么的问题，现在我们看看为什么的问题。

根据埃森哲的报告，财富 1000 强公司中，有四分之三受到新冠疫情导致的供应链中断的负面影响。根据麦肯锡的数据，物流和运输业受到不利影响，全球集装箱运输费率平均达到2019年水平的四到五倍，而某些特定市场的增幅甚至更高。

控制螺旋式上升的成本和提高短期投资回报率这两个问题令利益相关者和 C-level 执行官们面临越来越大的压力，他们考虑如何提高企业的弹性和敏捷性，同时平衡以此带来的风险。供应链结构已经发生了变化，由于各种因素，去年有 48% 的供应链变得更为复杂，这些因素包括：

### 地域扩张和更大的客户群

电子商务为企业提供了全球扩张的机会，但同时也意味着供应链将更加复杂。企业必须考虑如何在新的区域构建供应链以及如何在新市场中交付订单。
  
### 多样化的供应和新的采购策略

许多企业正在从传统的线性供应链转向更复杂的网络型供应链，以避免采购出现问题。毕马威表示，企业已经认识到需要确定更多的备选贸易伙伴来更好地组织其供应链，以防止出现供应链的单点故障。这无疑增加了业务的复杂性，包括供应商和潜在新服务提供商在内的多个相互关联的合作伙伴形成了一个复杂的供应商网络。

### 交货预期

客户的期望已经发生变化，现在需要提供实时订单状态和货物分发更新信息。正如 DHL 所强调的那样，物流正变得更加多样化。维持供应链稳定性的一个关键因素是业务的敏捷性，即能够快速适应变化，但这确实也表示供应链的管理将更加复杂。使用物流软件应该可以应对这一挑战，软件能有效地管理日益增加的复杂性，而无需费力的手动流程，并减少人员成本。

为了真正实现到物流自动化带来的益处，则供应链系统不能孤立运行。供应链依赖很多外部因素，必须作为企业更广泛的数字生态系统的一部分运行，以确保在面临内外部突发事件、市场条件和不断变化的战略时，能及时优化并做出改变。

企业在寻求降低成本和提高效率的过程中，都看到了端到端物流管理和流程自动化系统的价值，通过 API 技术，可以实现多个系统的互联互通，将更复杂的业务流程进行自动化。

## 变革的障碍

根据麦肯锡的一项研究，85% 的受访者在其供应链中使用了低效的数字技术，只有 14% 的受访者实现了持续的绩效改进。

尽管物流自动化有很多好处，但许多组织在做出积极变革时，往往面临许多障碍，包括：

### 缺少数字化战略

尽管自动化具有很多好处，但真正的数字化转型不仅仅是对一些业务流程进行自动化而已。成功的数字化转型要靠多部门合作，并考虑各种因素，包括优化业务流程以克服文化变革的阻力。数字化战略要逐渐成为业务战略的基础，而这一点，需要整个组织的协调。专注于一个方面而忽视其他方面可能会浪费时间、金钱和机会来实现真正的变革。

### 供应链的复杂性

供应链的复杂性将继续成为全球供应链巨头们关注的重大问题，尤其是在自动化方面。46% 的供应链巨头表示，很难整合整个供应链（从制造到最终客户交付）的全部功能。一方面，自动化是管理供应链复杂性的一种有效的方法，但与此同时，实施正确有效的自动化方案可能会增加复杂度和压力。例如，随着企业迁移到新技术，工作流程可能会变得支离破碎，因为由于业务连续性的需求，并非所有流程都可以同时变更。

### 见效慢

根据普华永道的估计，物流业的利润相对较低，息税前利润率一般在 1% 至 8% 之间。先进的物流和运输自动化通常需要大量的财务投资，且需要花费数千美元的年费来支持、维护和升级。新的解决方案上线，就需要不短的时间才能看到该笔投资的回报。

成功创造价值的公司从非常明确和具体的投资回报率目标开始，根据这些目标衡量他们的绩效，并不断迭代以达到目标。这样可以节省成本，减少效率低下，提高客户服务质量和生产力。

### 选择了错误的技术

现有硬件/软件环境与新技术难以兼容也可能导致实现自动化难以实现的问题。在已经非常复杂的技术环境中添加新的技术通常会比较头疼，此时应该考虑如何让新平台适应现有技术并帮助实现长期战略目标。

任何新的物流和运输软件解决方案的终极目标都应该是提高业务的整体生产力和盈利能力。

在决定实施任何新技术之前，应根据几个重要标准对其进行评估。

- 如何帮助提高企业的整体生产力和盈利能力
- 如何轻松地将其集成到现有 IT 环境中
- 如何帮助克服对软件供应商的依赖
- 新技术是否可扩展和可定制
- 如何帮助最大限度地降低成本

## 物流自动化软件

![运输管理系统（TMS）](https://cdn.abmcode.com/zh-cn/industry/_media/logistics/logistics_4.svg) {.center .size-4 .float-r}

### 运输管理系统（TMS）

运输管理系统（TMS）是物流自动化的基础。根据 Gartner 的说法，TMS 的范围包括所有运输方式的货运规划，货运评级和采购等流程；选择合适的路线和承运人；货运账单和付款管理。

其中许多流程可以由通用 ERP 系统或更大的供应链管理（SCM）系统部分覆盖，但是这些系统很笨重，并且难以管理复杂和动态变化的物流流程。这些系统缺乏敏捷性，无法集成到更大的工作流程中，也不能足够快地适应变化以满足业务需求。因此，根据行业研究，38% 的航运公司和 67% 的第三方物流公司表示他们正在投资运输管理系统。

运输管理系统可以通过自动化调度来提高规划效率。汽车运输中，零担运输（LTL）的数量正在增长，因此，在一个系统中比较多个承运人的费率的能力对于降低每百重量成本或每公里货运成本至关重要。在多式联运场景中，TMS 能在不同的货物交付利益相关者之间自动化工作流程，管理所有支持的行政流程。

TMS 可以立即为企业带来的另一个价值是降低违法的风险以及可能受到法律法规处罚和罚款的风险。自动化后台流程有助于确保每批货物都正确记录并合规。

![车队管理](https://cdn.abmcode.com/zh-cn/industry/_media/logistics/logistics_6.svg) {.center .size-4 .float-l}

### 车队管理

近年来，车队管理发生了重大变化。随着企业越来越关注汽车运输与“双碳”的问题，许多企业正在寻找监测和管理车队碳排放的方法。道路和车载传感器便是一种常见的监测方法，并为过程自动化创造了新的机会。

车队管理软件旨在通过自动化核心流程来改善车队运营，包括车辆的位置跟踪、维护调度、驾驶员管理和路线规划。此外，还可以帮助监测可能导致空气污染的排放，记录服务时间（HOS）以满足合规性并提高驾驶员的安全性。车队管理解决方案还可以处理车辆租赁/购买和保险续保等问题。

车队管理系统有助于企业实现更低的油耗、更低的维护成本、更优化的时间表以及更少的延误。

许多人认为只有运营数百辆车的大公司才有必要部署车队管理系统，这是一个常见的误区。重要的并不是车队的规模。只要核心运营流程依赖于运输，车队管理系统都可确保能最佳利用每项资产，并协助完成耗时的管理任务。这无形中提高了整体绩效，最终构筑企业的竞争门槛。

![调度管理](https://cdn.abmcode.com/zh-cn/industry/_media/logistics/logistics_7.svg) {.center .size-4 .float-r}

### 调度管理

很明显，新冠疫情对购买行为和电子商务的影响已经造成了永久性的转变，尤其是在对配送服务的需求方面。这个行业严重依赖劳动力。随着工作量空前增长，招聘和培训足够的人员以及管理大量派送工作人员变得越来越困难。

调度管理软件可用于管理车辆协调、设备和自动化工作流程，以完成订单或服务请求。还可以自动化人员管理，例如监督现场的司机或派送人员以及完成工作订单分配。

在调度中使用流程自动化可以改善客户服务并实现更高效的物流运营。

![供应链管理](https://cdn.abmcode.com/zh-cn/industry/_media/logistics/logistics_8.svg) {.center .size-4 .float-l}

### 供应链管理

供应链可视化管理软件为跟踪从原材料/组件到采购、分销、运输和交付给客户的整个物流周期提供了所有信息。根据业务类型，还可能包括售后服务。

供应链可视化管理解决方案用于管理供应链网络的复杂性，从而实现内部运营和外部合作伙伴之间的战略协作。

安永调查显示，超过 60% 的供应链高管表示，提高可视化是他们未来三年的首要任务。跟踪和追溯系统是供应链可视化的基础。支持在整个供应链中提供有关产品位置的实时信息，并能够将产品追溯到生产地点和使用什么材料。这在某些受监管的行业（如制药）中特别有用。

由于有效的业务运营需要更多的透明度和可见性，跟踪和追溯解决方案在其他行业中也变得很重要。

## 如何开始

![如何开始自动化](https://cdn.abmcode.com/zh-cn/industry/_media/logistics/logistics_9.svg) {.center .size-4 .float-r}

运输自动化的核心功能通常以整合货物和根据每个客户的需求计算最佳交付路线为主。最终目标是最大限度地提高每辆车和驾驶员的容量，降低复杂条件下的交付成本。

市场上有许多解决方案可以解决这个问题。运输管理系统通常支持所有运输方式，包括整车运输（FTL）、零担运输 （LTL） 以及这两者的混合。然而，在动态环境中，运输规划人员无法依靠传统的，简单的经验法则来优化通用 TMS 的中硬编码的运输类型。随着多式联运的发展，该行业需要新的解决方案，可以自动进行规划和优化，解决围绕这些流程产生的复杂性。

在规划方面，许多大型和小型公司仍然依靠 Excel 电子表格和运营经理的专业知识来完成这项工作。这些费力的手动流程需要巨大的工作量，并无法实现其他流程自动化。随着模型的规模和复杂性的增长，许多公司正在寻找一种摆脱对 Excel 依赖的方法，而这是自动化开始的最佳起点。
